资产报废管理

 

管理目标：

通过规范省、地市公司的资产报废流程，对资产的退出过程给予明确的指导，从而加强资产报废控制力度，提升资产报废操作的合规性、及时性、准确性，提高报废资产处置效率。

 

使用部门资产管理员收集汇总资产报废需求后，在EAM系统中发起报废申请并生成资产报废工单。

在EAM系统中审批实物使用部门提出的报废资产申请

 

在主菜单的“资产实物管理”模块，点击“资产报废管理”，显示三个子功能模块：

n         创建报废申请：启动资产报废流程。

n         报废单据查询：各类报废单据查询。

n         报废单据打印: 打印相关的报废单据。
